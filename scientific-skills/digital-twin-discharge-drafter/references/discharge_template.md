## 各章节书写要点

### 1. 入院情况 (Admission Status)

#### 1.1 主诉
- 简洁明了，20字以内
- 包含主要症状 + 持续时间
- 示例: "反复胸痛3天，加重1天"

#### 1.2 现病史
- 发病时间、诱因
- 主要症状特点
- 伴随症状
- 诊疗经过
- 一般情况

#### 1.3 入院查体
- 生命体征（T、P、R、BP）
- 一般状况
- 专科阳性体征

#### 1.4 辅助检查
- 入院前重要检查
- 入院后初步检查
- 关键异常指标

---

### 2. 诊疗经过 (Treatment Course)

#### 2.1 诊断过程
- 鉴别诊断思路
- 确诊依据

#### 2.2 治疗过程
- 药物名称、剂量、疗程
- 手术/操作记录
- 特殊治疗

#### 2.3 病情演变
- 治疗效果评价
- 并发症处理

---

### 3. 出院医嘱 (Discharge Orders)

#### 3.1 用药指导
- 药物名称、剂量、用法
- 疗程
- 注意事项

#### 3.2 饮食与生活方式
- 饮食类型（低盐/低脂/糖尿病饮食等）
- 活动限制
- 休息要求

#### 3.3 复查建议
- 复查项目
- 复查时间

#### 3.4 随诊指导
- 随诊时间
- 随诊科室
- 紧急情况处理

## 常见疾病出院医嘱示例

### 心血管疾病

```
出院医嘱:
1. 出院带药:
   - 阿司匹林 100mg 每日一次 早餐前服用
   - 阿托伐他汀 20mg 每晚一次
   - 美托洛尔 25mg 每日两次

2. 饮食: 低盐低脂饮食，戒烟限酒

3. 活动: 适当有氧运动，避免剧烈活动

4. 复查: 出院后1周复查凝血功能，1月复查血脂、肝功能

5. 随诊: 心内科门诊定期随访，如有胸痛加重立即就诊
```

### 呼吸系统疾病

```
出院医嘱:
1. 出院带药:
   - 吸入用布地奈德福莫特罗 1吸 每日两次
   - 孟鲁司特钠 10mg 每晚一次

2. 饮食: 清淡易消化饮食，多饮水

3. 活动: 避免接触过敏原，雾霾天减少外出

4. 复查: 出院后2周复查肺功能

5. 随诊: 呼吸科门诊随访，指导吸入剂正确使用
```

### 消化系统疾病

```
出院医嘱:
1. 出院带药:
   - 奥美拉唑 20mg 每日一次 早餐前
   - 铝碳酸镁 1g 每日三次 餐后咀嚼

2. 饮食: 清淡软食，少食多餐，避免辛辣刺激

3. 活动: 规律作息，避免过度劳累

4. 复查: 出院后4-6周复查胃镜

5. 随诊: 消化内科门诊随访
```

## 书写规范要求

### 一般要求
1. 字迹工整，语句通顺
2. 使用医学术语，表述准确
3. 内容完整，重点突出
4. 时间记录精确到日

### 格式要求
1. 使用黑色或蓝黑色墨水
2. 不得涂改，如需修改须签名并注明日期
3. 各栏目填写完整，不留空项
4. 医师签名清晰可辨

### 内容要求
1. 真实反映诊疗过程
2. 诊断名称规范（使用ICD编码）
3. 用药信息完整准确
4. 医嘱具体可操作

## 附录: 常用缩写

| 缩写 | 全称 | 含义 |
|------|------|------|
| qd | quaque die | 每日一次 |
| bid | bis in die | 每日两次 |
| tid | ter in die | 每日三次 |
| qid | quater in die | 每日四次 |
| po | per os | 口服 |
| iv | intravenous | 静脉注射 |
| im | intramuscular | 肌肉注射 |
| ac | ante cibum | 饭前 |
| pc | post cibum | 饭后 |
| prn | pro re nata | 必要时 |
