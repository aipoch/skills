# Chemical Storage Sorter

Chemical compatibility organizer.

## Use Cases
- New lab setup
- Safety inspections
- Chemical inventory
- EHS compliance

## Parameters
- `chemical_list`: Inventory
- `storage_type`: Cabinet/fridge

## Returns
- Compatibility groups
- Cabinet assignments
- Incompatibility warnings
- Segregation rules

## Example
Acids separate from bases, oxidizers from organics
